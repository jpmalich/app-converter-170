Design system preview · Estimate Editor + Dashboard

Evolving the industrial system — not repainting it

The current black-and-safety-orange language is the right direction for a pro trade tool. The redesign keeps it and modernizes the delivery: soft elevation, a consistent radius, real interactive states, and per-company theming — while retiring the open accessibility debt. Same screens, restyled. Orange stays.

Company theme Theme applies to the After column — the same screen, rebranded per contractor company from one brand color.

Soft elevation

A 3-step shadow scale replaces flat hard edges — cards lift, dialogs sit above.

Consistent radius

A 7px radius token on cards, inputs, buttons & pills; the sell-bar keeps its sharp industrial edge.

Real states

Hover, focus-visible ring, loading & disabled on every control; status as pills, not just text.

Per-company brand

A company theme is a generated token block from one brand color, contrast-validated before save.

Dashboard

Contractor landing — pipeline at a glance, then the estimate list.
